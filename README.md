# Super-trunfo-N-vel-Novato
Criando as Cartas do Super Trunfo   Neste primeiro nível, seu objetivo é construir a base do nosso jogo Super Trunfo de Países: um sistema para cadastrar as cartas com informações sobre as cidades. Você vai praticar a leitura de dados do usuário, o armazenamento em variáveis e a exibição dessas informações na tela.
scanf(" %[^\n]", cidade) permite ler nomes com espaços (ex: Rio de Janeiro).
O espaço antes de %c (" %c") evita problemas com ENTER pendente no buffer.
%.2f formata números com duas casas decimais.
Nenhuma estrutura de repetição (for, while) ou decisão (if) foi usada, conforme exigido.
